# RepopTracker Log Reporter

Desktop app that watches your EQ log file and reports slain events to the RepopTracker server. For non-technical users: download, install, enter the password you were given and your log path, then leave it running.

## For end users

1. Get the password from your admin.
2. Install and run the app (double-click the exe).
3. Enter the password and the full path to your EQ log file (e.g. `C:\...\eqlog_Character_Project1999.txt`).
4. Click "Save and start". Leave the app running while you play.
